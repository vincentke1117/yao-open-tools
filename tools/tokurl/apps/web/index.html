把长链接，一键缩短

TokURL 是一个开源、自部署的短链工具，支持极速生成、全球跳转、实时统计、二维码、二次编辑和团队用户管理。

长链接 https://example.com/campaign/landing-page?utm_source=wechat&utm_campaign=launch
→
短链接 https://your-domain.com/aB12c

毫秒级跳转

Redis 缓存命中后直接跳转，访问统计异步写入，外部访问体验稳定快速。

实时流量统计

按短链统计点击趋势、来源、设备和最近访问事件，支持查看全部链接数据。

开源自部署

Web、API、worker、Postgres 和 Redis 多容器部署，配置环境变量即可上线。
